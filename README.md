# APlane SDKs Have Moved

The APlane SDK packages have moved to the `aplanesdk` package and repository.

Use the new packages for new integrations:

```bash
npm install aplanesdk algosdk
pip install aplanesdk
```

The SDK source now lives at:

https://github.com/aplane-algo/aplanesdk

The previous `aplane` SDK packages are no longer maintained from this
repository. They are kept only as a historical package name.
